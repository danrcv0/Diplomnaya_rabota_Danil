# 🧠 Psychological Support Web App


Веб-приложение для сервиса психологической помощи.  
Проект демонстрирует работу с Flask, авторизацией, базой данных и административной панелью.

## 🚀 Основной функционал

### 👤 Пользователь:
- Заполнение заявки на консультацию
- Авторизация (через email)
- Оставление отзывов (после посещения встречи)
- Просмотр информации на главной странице

### ⭐ Отзывы:
- Возможность оставить отзыв только после подтвержденной встречи
- Отображение последних отзывов на главной странице

### 🔐 Авторизация:
- Подтверждение через email (код)
- Работа с сессиями

### 🛠 Админ-панель:
- Управление пользователями
- Контроль заявок
- Интерфейс администратора

## ⚙️ Технологии

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML / CSS / JavaScript

---


## 🗄 База данных

- Используется SQLite (app.db)
- Создаётся автоматически при запуске

---

## 📬 Основные эндпоинты

| Метод | URL              | Описание |
|------|-----------------|----------|
| GET  | /               | Главная страница |
| POST | /submit         | Отправка заявки |
| POST | /submit_review  | Отправка отзыва |

---

## ⚠️ Особенности

- Отзыв можно оставить только если встреча завершена
- Используются Flask Blueprint'ы
- Есть базовая обработка ошибок

---

## 🔒 Безопасность

- Статический secret_key
- Нет CSRF защиты
- Упрощённая авторизация
